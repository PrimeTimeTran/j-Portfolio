## Quick Start

```terminal
  gem install bundler jekyll

  jekyll new my-awesome-site

  cd my-awesome-site

  bundle exec jekyll serve
```

## Tips

Hot reload on changes

```terminal
bundle exec jekyll serve --livereload
```
